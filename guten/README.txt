== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Upgrade Notice ==

Purchase Guten Premium: https://gutentheme.org/go/purchase/ and after purchase you will be able to download the Premium theme as a zip file.

Simple Upgrade Instructions:
1. Delete the Guten free theme version
OR
1. Install the Easy Theme and Plugin Upgrades Plugin: https://wordpress.org/plugins/easy-theme-and-plugin-upgrades/
2. Download the Guten Premium zip file and install as a zip at Dashboard -> Themes -> Add New -> Upload Theme.

Needing help on using the theme? See our Documentation: https://gutentheme.org/documentation/


Guten's Customizer Settings:
---------------
The theme settings are built into the WordPress Customizer using "Customizer Library" by Devin Price, situated in /customizer/.

The Guten upgrade situated in /upgrade/ displays the features that the premium version includes.
Guten version is licensed under GPL 2.0+

All setting are self explanatory or have notes explaining what they do or how to use the theme settings.
View the theme settings under "Appearance" -> Customize.
